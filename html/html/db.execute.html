db.execute

NAME

db.execute - execute any SQL statement
(GRASS Database Program)

SYNOPSIS

db.execute [driver=name] [database=name] [location=name] [input=filename]

Parameters:

driver
Driver name
database
Database name
location
Database location
input
Filename with sql statement
