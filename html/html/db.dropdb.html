db.dropdb

NAME

db.dropdb - remove a database
(GRASS Database Program)

SYNOPSIS

db.dropdb driver=name database=name [location=name]

Parameters:

driver
Driver name
database
Database name
location
Database location

NOTE

Module does not work with ODBC driver
