d.barscale

NAME

d.barscale - displays a barscale on GRASS monitor
(GRASS Display Program)

SYNOPSIS

d.barscale [-m] [bcolor=name] [tcolor=name] [at=x,y]

Flag:

-m
Use mouse to interactively place scale

Parameters:

bcolor
Color used for the background
Options: red, orange, yellow, green, blue, indigo, white, black, brown, magenta, aqua, gray, grey
Default: black
tcolor
Color used for the text
Options: red, orange, yellow, green, blue, indigo, white, black, brown, magenta, aqua, gray, grey
Default: white
at
The screen coordinates for top-left corner of label
Options: 0-100
Default: 0.0,0.0
