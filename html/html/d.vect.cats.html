d.vect.cats

NAME

d.vect.cats - tool for viewing vector maps with labels. (GRASS vector program)

SYNOPSIS

d.vect.cats [-f] map=name [color=name] cat=value[,value,...]

DESCRIPTION

d.vect.cats displays a vector map (vector binary file), mapname in the current graphics window. This window must be a map window, or an unclaimed window. Options are

Parameters:

map=mapname
Name of vector map to be displayed
cat=value
Category value to be displayed
color=color
Color desired for drawing map
Default: white

The window type is set to map if not already.
