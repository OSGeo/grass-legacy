NAME

r3.in.v5d - Imports files in v5d format to G3D format.
(GRASS 3D Program)

SYNOPSIS

r3.in.v5d input=name output=name [nv=name] [type=name] [precision=name] [compression=name] [tiledimension=name]

DESCRIPTION

Vis5D is a system for interactive visualization of large 5-D gridded data sets such as those produced by numerical weather models. One can make isosurfaces, contour line slices, colored slices, volume renderings, etc of data in a 3-D grid, then rotate and animate the images in real time. There's also a feature for wind trajectory tracing, a way to make text anotations for publications, support for interactive data analysis, etc.
r3.in.v5d imports 3-dimensional files (i.e. the v5d file with 1 variable and 1 time step). Otherwise, only first variable and timestep from 4/5D v5d file will be imported.

Parameters:

input
Path and name of v5d file to be imported
output
Name of the G3D output raster map
nv
String representing NULL value data cell (use 'none' if no such value)
Default: none
type
Data type used in the output file
Options: default, double, float
Default: default
precision
Precision used in the output file
Options: default, max, 0-52
Default: default
compression
The compression method used in the output file
Options: default, rle, none
Default: default
tiledimension
The dimension of the tiles used in the output file
Default: default
