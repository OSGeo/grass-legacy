GRASS 5.4.x Reference Manual

This reference manual details the use of modules distributed with Geographic Resources Analysis Support System (GRASS), an open source (GNU GPL'ed), image processing and geographic information system (GIS).

NOTE: Some modules listed here are not yet available in GRASS 5.

GRASS development has been conducted since 1995 at USACERL, from 1997 onwards by the international GRASS Development Team headquartered at ITC-Irst (SSI/MPA), Trento, Italy.

The manual pages explain the capabilities, syntax, and use of current GRASS 5.3.x modules. Sections contain entries which are ordered alphabetically by module name. Each entry contains:

  • the module name
  • a brief description
  • a synopsis of files associated with the module
  • a full description of the module's capabilities and uses
  • a summary of the command-line operations associated with the module
  • a list of associated or related module(s)
  • a list of references concerning the method (if required)
  • a list of module's authors
Each entry may optionally include a description of the module's operation in interactive mode, and a discussion of "bugs," limitations, or special notes of interest.

Throughout the manual, module input to be explicitly entered by the user is printed in bold text. User-specified input (i.e., variables set by the user) is set in italic type. Boxed text shows information output to the user's terminal screen.
