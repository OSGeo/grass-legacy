DESCRIPTION

Calculates basic univariate statistics for numeric attributes in a data table. It will calculate minimum, maximum, range, mean, standard deviation, variance, coefficient of variation, quartiles, median, and 90th percentile. It is closely based on r.univar.sh and uses db.select to create a list values for statistical calculations. This script could be replaced by an updated v.univar.

(GRASS Shell Script)

NOTES

If the database and driver are not specified, the module will use the default values set in db.connect.

EXAMPLE

g.region rast=elevation.10m -p
v.random out=samples n=100
v.db.addtable samples col="heights double precision"
v.what.rast samples rast=elevation.10m col=heights
v.db.select samples

v.univar.sh samples col=heights
