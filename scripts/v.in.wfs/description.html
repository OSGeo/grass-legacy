DESCRIPTION

v.in.wfs imports OGC WFS maps (Web Feature Service) from external servers.

EXAMPLES

Import of world cities with population > 1 million: 

# run in Latitude-Longitude location (EPGS code 4326):

# download geonam_gt1M layer:
v.in.wfs \
 wfs="http://mapserver.flightgear.org/ms?Service=WFS&request=GetFeature&version=1.0.0&typename=geonam_gt1M" \
 output=geonam_gt1M

REQUIREMENTS

The "curl" http download program must be installed. The OGR library on the system needs to be compiled with Xerces C++ XML Parser support (for GML).
