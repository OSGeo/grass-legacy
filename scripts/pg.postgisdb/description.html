NAME

pg.postgisdb

DESCRIPTION

pg.postgisdb is used to create and destroy PostGrass databases. All options that are not available in primary options scope will be ignored. Only Postmaster can run this script to create or delete a new database (use -U to define database owner)

EXAMPLE

pg.postgisdb -c botswanaUTM34 -h pgserver.itc.it -U postgres -G ./

NOTES

PostGIS must be installed for this to work.
The PostGIS homepage can be found at http://postgis.refractions.net
